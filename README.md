# 🌍 CarbonIQ – AI-Powered Carbon Footprint Awareness Platform

CarbonIQ helps individuals understand, track, and reduce their environmental impact through AI-powered sustainability insights, carbon footprint analytics, smart recommendations, and interactive what-if simulations.

## 🚀 Problem Statement

Many people want to live sustainably but lack visibility into how their daily activities contribute to carbon emissions. CarbonIQ bridges this gap by transforming everyday actions into measurable environmental impact and providing personalized guidance for reducing footprints.

## Known Limitations

- Gemini API may temporarily use fallback mode due to quota limits.
- Emission values are estimated using predefined factors.
- Forecasting accuracy improves with more user history.
